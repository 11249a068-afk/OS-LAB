➡Ex No: 1a — Installation of Windows Operating System
➡Aim
To install the Windows Operating System on a personal computer and understand the basic steps involved in OS installation, including disk partitioning, file system selection, and system configuration.

➡Procedure
Obtain a bootable Windows installation media (USB/DVD) with a valid Windows ISO file.
Insert the bootable media into the system and restart the computer.
Enter the BIOS/UEFI setup and set the boot priority to USB/DVD.
Save the BIOS settings and reboot the system.
When the Windows setup screen appears, select the language, time, and keyboard settings, then click Next.
Click Install Now and enter the product key if prompted, or select Skip.
Choose the appropriate Windows edition and accept the license agreement.
Select Custom Installation to install Windows on a specific partition.
Create, delete, or format disk partitions as required, then select the target partition.
Allow the system to copy Windows files and complete the installation automatically.
After installation, configure basic settings such as username, password, date, time, and network.
Once setup completes, the Windows desktop appears — indicating a successful installation.
➡Result
The Windows Operating System was successfully installed on the system and basic system configuration was completed as per the given procedure.
